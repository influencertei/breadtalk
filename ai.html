AI가 판단하는 빵 상태 4가지

빵톡 AI는 Google의 Teachable Machine을 기반으로 학습된 전용 모델입니다. 직접 촬영한 건강빵 이미지를 학습 데이터로 사용해 4가지 상태를 구분합니다.

✅
완벽하게 구워진 빵

겉면이 황금빛 갈색으로 고르게 구워지고, 크러스트가 적당히 바삭하며 속은 촉촉하게 완성된 상태입니다. 손가락으로 가볍게 두드렸을 때 '통통' 소리가 납니다.

⚠️
덜 구워진 빵

겉면 색이 옅고 눌렀을 때 탄력이 없으며 속이 아직 익지 않은 상태입니다. 오븐 온도를 확인하고 5~10분 추가로 구워야 합니다.

🔥
과하게 구워진 빵

겉면이 진한 갈색 또는 검게 타기 시작한 상태입니다. 수분이 날아가 딱딱해지고 쓴맛이 날 수 있습니다. 다음번엔 온도를 10~15°C 낮추세요.

🍞
발효 중인 반죽

아직 오븐에 넣기 전 발효 단계의 반죽 상태입니다. 반죽이 충분히 부풀었는지 확인하고, 손가락으로 눌렀을 때 천천히 복원되면 발효 완료입니다.

기술 소개

빵톡 AI 분석 도구, 어떻게 만들었나요?

Google Teachable Machine 기반

빵톡 AI 분석 도구는 Google의 Teachable Machine 플랫폼을 활용해 만들었습니다. Teachable Machine은 코딩 지식 없이도 이미지 인식 AI 모델을 직접 학습시킬 수 있는 도구입니다.

  • 빵톡 채널의 실제 건강빵 사진을 학습 데이터로 사용
  • TensorFlow.js 기반으로 브라우저에서 직접 실행
  • 카메라 또는 사진 업로드 두 가지 방식 지원
  • 사진이 서버로 전송되지 않아 개인정보 안전

실제 활용 팁

AI 분석 도구를 가장 잘 활용하는 방법을 알려드립니다. 좋은 사진일수록 정확도가 높아집니다.

  • 빵 윗면 전체가 보이도록 위에서 찍어주세요
  • 밝은 곳에서 촬영할수록 결과가 정확합니다
  • 빵 이외의 배경이 적을수록 좋습니다
  • 결과는 참고용이며, 최종 판단은 직접 확인하세요

자주 묻는 질문

AI 분석 도구 FAQ

Q

사진이 서버에 저장되나요?

아니요. 이 도구는 브라우저 내에서만 작동하며 사진이 외부 서버로 전송되거나 저장되지 않습니다. 개인정보 걱정 없이 사용하실 수 있습니다.

Q

정확도가 얼마나 되나요?

학습 데이터 기준으로 약 85% 이상의 정확도를 보입니다. 다만 촬영 환경(조명, 각도)에 따라 결과가 다를 수 있으니 참고용으로 활용하세요.

Q

어떤 빵에 사용할 수 있나요?

통밀빵, 식사빵, 깜빠뉴, 포카치아 등 빵톡에서 소개하는 건강빵 종류에 최적화되어 있습니다. 케이크나 과자류는 인식 정확도가 낮을 수 있습니다.

Q

모바일에서도 사용할 수 있나요?

네. 스마트폰 브라우저에서도 동작합니다. 사진 업로드 모드에서는 갤러리 사진 또는 즉시 촬영 중 선택할 수 있습니다.
